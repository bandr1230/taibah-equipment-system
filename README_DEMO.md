# نسخة العرض للمديرين

هذه النسخة مخصصة للعرض التنفيذي فقط. تعمل ببيانات تجريبية محلية داخل المتصفح ولا تعتمد على Supabase أثناء العرض.

## الدخول السريع

- مدير النظام: `admin / 123`
- إدارة التجهيزات: `equipment / 123`
- مسؤول كلية الصيدلة: `pharmacy / 123`
- مشاهد التقارير: `reports / 123`

## سيناريو عرض مقترح

1. افتح اللوحة التنفيذية واعرض المؤشرات: القطاعات، الأصناف، الأصناف تحت الحد، والطلبات النشطة.
2. انتقل إلى طلبات الاحتياج واعرض اختلاف الحالات: بانتظار القطاع، بانتظار إدارة التجهيزات، معاد للتعديل، ومعتمد.
3. افتح شواهد الاحتياج لشرح ربط الطلب بالمقرر أو الدليل والكمية المقدرة والعجز.
4. افتح طلب الدعم بين القطاعات لشرح الاستفادة من المخزون المتاح لدى كلية أخرى.
5. من اللوحة التنفيذية اشرح "مساعد التحليل": الأكثر صرفًا، الأعلى احتياجًا حسب الاستخدام، والأكثر طلبًا.
6. افتح صفحة "المحلل الذكي" لعرض الأسئلة الجاهزة، السؤال الحر، التوصيات التلقائية، ونموذج تجهيز الربط لاحقًا مع API ذكاء اصطناعي آمن.
7. افتح التقارير ثم تقرير الإدارة العليا، وبعدها جرّب استخراج PDF.

## ملاحظات تشغيل

- زر "إعادة ضبط العرض" يعيد البيانات التجريبية إلى الحالة الأصلية.
- تصدير Excel يحتاج تحميل مكتبة `XLSX` من الإنترنت كما في `index.html`.
- هذه النسخة ليست إعداد إنتاج رسمي، لكنها تعرض تجربة إدارية مكتملة ومقنعة.
