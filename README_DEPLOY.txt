نسخة العرض للمديرين - تعليمات نشر مختصرة:

1) ارفع الملفات كما هي إلى GitHub أو Vercel.
2) نفذ Redeploy مع Clear Cache إذا كانت هناك نسخة قديمة.
3) افتح الصفحة الرئيسية ثم استخدم:
   admin / 123
4) النسخة تعمل محليًا ببيانات تجريبية ولا تتصل بـ Supabase أثناء العرض.
5) عند الحاجة لإعادة السيناريو إلى بدايته اضغط "إعادة ضبط العرض".

المحلل الذكي يعمل حاليًا بقواعد محلية. عند الربط لاحقًا مع ذكاء اصطناعي حقيقي، اضبط SMART_ANALYST_API_ENDPOINT على رابط backend آمن ولا تضع مفاتيح API داخل ملفات الواجهة.

ملاحظة: تصدير Excel يعتمد على مكتبة XLSX المحملة من CDN في index.html.
