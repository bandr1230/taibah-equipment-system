التعديلات المنفذة في النسخة v5.8

1) البنية التنظيمية
- إضافة مستوى: قسم رئيسي بين القطاع والقسم الفرعي.
- الإبقاء على الترميز للقطاع والقسم الفرعي فقط.
- تحديث إدارة المستخدمين بحيث حقل القسم = القسم الرئيسي.
- إضافة إدارة للمواقع ضمن صفحة القطاعات والأقسام والترميز.

2) المواقع
- إدراج مواقع كلية الصيدلة المستخلصة من ملف القاعات ضمن قاعدة البيانات الافتراضية.
- يمكن اختيار الموقع من القائمة أو كتابته يدويًا.

3) الأصناف والحركات
- إضافة القسم الرئيسي إلى الأصناف والحركات والتقارير.
- تعديل نماذج إضافة/تعديل الصنف لتشمل القسم الرئيسي.

4) طلبات الاحتياج
- تحديث نموذج رفع الاحتياج ليتوافق مع جدول الكميات:
  ERP, الفئة/القسم الفرعي, البند, وحدة القياس, وصف البند, المواصفات,
  منتج من القائمة الإلزامية, الرمز الإنشائي, بند متماثل,
  المبررات, ذكر العلامة التجارية, مبرر ذكر العلامة التجارية,
  كميات السنة الأولى/الثانية/الثالثة, رقم أمر الاحتياج,
  وطريقة الإرسال حسب القسم الرئيسي أو الفرعي.
- دعم جلب بيانات سابقة عند إدخال ERP مطابق.
- تحديث مسار الاعتماد إلى:
  منشئ الطلب -> اعتماد مسؤول القطاع -> إدارة التجهيزات
  مع دعم الإرجاع للتعديل وإضافة ملاحظة.

5) شواهد الاحتياج
- تمكين إضافة أكثر من شاهد في نفس عملية الإدخال قبل الحفظ النهائي.
- حفظ كل شاهد كسجل مستقل مرتبط بنفس طلب الاحتياج.

6) التقارير
- تحديث تقارير الأصناف والحركات والاحتياج لتشمل القسم الرئيسي.

ملاحظة مهمة
- تم تغيير مفتاح التخزين المحلي إلى إصدار جديد v5.8 لتفادي تعارض البيانات القديمة مع البنية الجديدة.
